# Synthetic FIR Samples

These images are generated exclusively for the PRAHARI prototype demo video.
They contain fictional identifiers and must not be represented as genuine police
records. Each sample is visibly marked as a synthetic, non-official document.

Recommended demo sequence:

1. Upload the clean English scan and select **English**.
2. Upload the bilingual Kannada sample and select **Kannada**.
3. Upload the phone-photo sample with **Auto-detect** to demonstrate scan handling.

PRAHARI currently returns an operator-review preview and does not persist the
uploaded record. Production OCR/VLM extraction remains an integration adapter.
