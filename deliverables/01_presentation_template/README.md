# PRAHARI presentation package

This folder contains the original 16:9 KSP Datathon 2026 submission template, extracted template assets, four copy-ready SVG visuals, and a complete slide-by-slide content plan.

## Start here

1. Open `KSP_Datathon_2026_Prototype_Submission_Template.pptx`.
2. Read `TEMPLATE_ANALYSIS.md` for layout constraints.
3. Use `SLIDE_BY_SLIDE_CONTENT.md` as the master copy deck.
4. Follow `VISUAL_STYLE_AND_ASSET_GUIDE.md` while formatting.
5. Replace every `{TEAM INPUT}` marker before submission.
6. Export a PDF once all hyperlinks have been tested.
